# Shopping Agent

Build an optimized shopping list from the current meal plan and send it via Telegram.

## Commands

### /shopping list

1. Load the current meal plan:
```bash
sqlite3 /data/workspace/pantry.db "SELECT plan_json FROM meal_plans ORDER BY created_at DESC LIMIT 1;" 2>/dev/null
```

2. Load fridge contents to remove already-owned items:
```bash
sqlite3 /data/workspace/pantry.db "SELECT item FROM fridge;" 2>/dev/null
```

3. Load best known prices:
```bash
sqlite3 /data/workspace/pantry.db "SELECT item, store, MIN(price) as price, unit FROM prices GROUP BY item;" 2>/dev/null
```

4. Display grouped by store (cheapest options first):
```
Shopping List — Week of [date]

Aldi:
  • pasta 500g — €0.89
  • eggs 10-pack — €1.79
  • milk 1L — €0.89

Lidl:
  • chicken breast 600g — €5.09
  • ground beef 500g — €4.29

Rewe:
  • salmon fillet 400g — €6.99

Estimated total: €19.94

Run /shopping send to send to Telegram
```

If no plan exists: "No meal plan found. Run /meals plan or /plan weekly first."

### /shopping send

Build the Telegram message and send it:

```json
{
  "action": "send",
  "channel": "telegram",
  "to": "$TELEGRAM_CHAT_ID",
  "message": "*Shopping List*\n\n*Aldi*\n• pasta 500g — €0.89\n• eggs — €1.79\n\n*Lidl*\n• chicken — €5.09\n\n*Estimated total: €[amount]*\n\n_Generated by OpenClaw Meal Planner_"
}
```

On success: "Shopping list sent to Telegram! ([N] items)"
On failure: Show the list in chat and explain: "Telegram not configured — make sure TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID are set in Railway env vars."

Also optionally post to Discord:
```json
{
  "action": "send",
  "channel": "discord",
  "to": "channel:$DISCORD_CHANNEL_ID",
  "message": "**Shopping List posted!** Check Telegram for the full list."
}
```

### /shopping optimize [budget]

1. Load shopping list and calculate total (as in /shopping list)
2. Compare to budget (default: €80)

If within budget:
```
Shopping list fits within €[budget] budget.
Estimated: €[total] — you're under by €[savings]!
```

If over budget, suggest substitutions:
```
Estimated €[total] exceeds €[budget] budget.

Consider substituting:
• salmon fillet €6.99 → canned tuna €1.49 (saves €5.50)
• chicken breast €5.09 → chicken thighs €3.99 (saves €1.10)

Adjusted estimate: €[new_total]
```

Always suggest cheaper store alternatives (Aldi/Lidl vs Rewe).

## Notes

- Group by store to make shopping efficient (one stop per store)
- Always show the estimated total prominently
- If Telegram env vars are missing, display the list in chat as fallback
- Keep list concise — 15 items max per Telegram message for readability
